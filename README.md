# PCDream
Prostate Cancer Dream Chanllenge code.
## Requirements
1. Use R 3.0 and please ensure that you have installed R package 'survival','randomForestSRC','caret'.
2. Make sure challenge data (CoreTable_leaderboard.csv, CoreTable_training.csv, CoreTable_validation.csv) is in your work directory.
3. Run this script and you will get three csv files in your work directory: finalq1a.csv, finalq1b.csv and finalq2.csv
